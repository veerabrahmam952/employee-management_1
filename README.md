# Employee Management System

## Overview
The Employee Management System is a web-based application designed to manage employee data efficiently. It provides features for adding, updating, deleting, and viewing employee details.

## Features
- Add new employees
- Update employee information
- Delete employee records
- View employee details
- Search and filter employees

## Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- A modern web browser

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/veerabrahmam952/employee-management.git
    cd employee-management
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm run start
    ```

## Usage
- Access the application at `http://localhost:4200` in your browser.
- Use the navigation menu to manage employees.

## Testing
Run the test suite using:
```bash
npm test
```

## Deployment
1. Build the application:
    ```bash
    npm run build
    ```

2. Deploy the `build` folder to your hosting provider.
